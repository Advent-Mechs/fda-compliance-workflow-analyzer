# FDA Compliance Workflow Analyzer

## Problem Statement
In regulated food manufacturing environments, 
compliance review workflows are often manual, 
spreadsheet-dependent, and time-intensive. 
This project simulates an automated pipeline 
that ingests FDA pesticide residue monitoring 
data, applies compliance logic, flags anomalies, 
and visualizes outcomes in an interactive 
dashboard — converting work that currently takes 
hours into a process that runs in seconds.

## Why This Project
If you could free up the cognitive load of your team by dismantling manual processes, and adopting automation, would you do it?
The answer is usually a resounding "yes". I want to lend a helping hand to this effort. This is my best shot at it. 

## Tech Stack
- Python (pandas, NumPy, matplotlib)
